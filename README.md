# Excel Viewer Frontend

A frontend application for visualizing excel documents.

## Technologies

- React (Vite)
- TypeScript
- Tailwind CSS

## Installation and Setup

1. Install dependencies:

```sh
npm install
```

2. Start the development server:

```sh
npm run dev
```

3. Build for production:

```sh
npm run build
```

4. Linting

```sh
npm run lint
```
